# ObserverPatternWeatherApp
Implemented a Weather Monitoring System in C# using the Observer Pattern. Multiple observers (Mobile App, LED Display, Website) subscribe to a Weather Station and receive automatic temperature updates. Demonstrates loose coupling, scalability, and dynamic subscriptions.
